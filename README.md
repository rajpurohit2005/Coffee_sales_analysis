Project Overview

This project performs an in-depth analysis of coffee shop sales data using Python.
The analysis focuses on identifying sales trends, customer behavior, peak business hours, payment preferences, and revenue-generating coffee products through advanced visualizations and business insights.

The project uses:

Python
Pandas
NumPy
Matplotlib
Features
Data Cleaning and Preprocessing
Revenue Analysis by Coffee Type
Daily Sales Trend Analysis
Monthly Revenue Analysis
Weekly Sales Analysis
Peak Hour Identification
Payment Method Distribution
Repeat Customer Analysis
Moving Average Trend Visualization
Revenue Contribution Analysis
Business Insights and Summary
Technologies Used
Python

Analysis Performed
1. Top Selling Coffee Types

Analyzes which coffee products generate the highest revenue.

2. Daily Sales Trend

Shows daily revenue fluctuations using line graphs.

3. Monthly Revenue Analysis

Identifies high-performing months.

4. Payment Method Analysis

Displays customer payment preferences using pie charts.

5. Weekly Sales Analysis

Determines the best sales day of the week.

6. Hourly Sales Analysis

Identifies peak business hours.

7. Repeat Customer Analysis

Finds loyal customers with frequent purchases.

8. Moving Average Trend

Displays long-term sales trends using moving averages.

9. Revenue Contribution Analysis

Shows product-wise contribution to total revenue.
